# 傻瓜版技能使用卡

你以后几乎只需要记住这一句：

`用 $skill-home 帮我选`

## 最省事的 6 句

```text
用 $skill-home 帮我选
用 $skill-home 看看这件事该用哪个 skill
用 $skill-home 我不知道该用什么
用 $skill-home 先推荐，再问我确认
用 $skill-home 看看我现在装了哪些 skill
用 $skill-home 找找有没有适合这个需求的 skill
```

## 直接套用模板

```text
用 $skill-home 我想研究一下{产品/公司/概念}
用 $skill-home 我想写一篇{公众号文章/长文}
用 $skill-home 帮我看看这个{业务/商业模式/方向}
用 $skill-home 审一下这个 AI 产品需求
用 $skill-home 整理一下这个项目文档
用 $skill-home 我想继续上次那个商业诊断
```

## 它会怎么工作

1. 先听懂你要做什么。
2. 自动挑一个最适合的 skill。
3. 先问你要不要用它。
4. 你同意后才真正调用。

## 你会看到的典型回复

```text
我建议用 $hv-analysis，因为你这是深度研究，不是普通聊天。要我现在用它继续吗？
```

```text
我建议用 $dbs，因为你现在的问题还比较模糊，先让它帮你判断方向最省事。要我现在用它继续吗？
```

```text
我建议用 $khazix-writer，因为你是想把素材写成长文。要我现在用它继续吗？
```

## 只想看现在装了什么

```text
用 $skill-home 看看我装了哪些 skill
```

## 想找新 skill

```text
用 $skill-home 看看有没有适合做{你的任务}的 skill
```

如果本地没有合适的，它应该优先推荐 `$find-skills`。
