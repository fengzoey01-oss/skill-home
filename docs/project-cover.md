# 项目封面介绍

## 标题

skill-home：给 Codex Skills 做一个真正能用的入口

## 一句话介绍

一个帮助用户“先选 skill，再决定是否继续”的技能首页，让多 skill 系统从“能用”变成“好用”。

## 短简介

当你装了越来越多 Codex skills 之后，真正的问题不再是“有没有能力”，而是“我现在该用哪个”。

`skill-home` 就是为这个问题做的。

它让用户不用记 skill 名，不用理解内部路由，只要用自然语言说出目标，就能先得到推荐，再决定是否继续调用。

## 适合放在仓库首页 / 分享页的文案

很多 AI 系统的问题，不是能力不够，而是入口太复杂。

`skill-home` 不是再增加一个 skill，而是给整个 skill 系统补一个首页：

- 说人话就能开始
- 自动推荐最合适的 skill
- 调用前先征求确认
- 后续新增 skill 也能自动纳入视野

如果你装了越来越多 skills，但不想每次都先想“该用哪个”，这个项目就是为你准备的。
