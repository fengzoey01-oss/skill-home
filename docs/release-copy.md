# 发布配套文案

## 朋友圈版

最近用 Codex 封装了一个我自己特别想要的小东西，叫 `skill-home`。

它不是帮你多做一件事，而是帮你少想一件事：
当你装了越来越多 skills 之后，每次开始前最烦的那个问题其实是，“我现在到底该用哪个？”

所以我做了一个技能首页。

你不用记 skill 名，不用理解内部路由，只要说人话，比如：

`用 $skill-home 帮我选`

它就会先推荐最适合的 skill，再问你要不要继续。

我觉得这玩意儿最有价值的地方不是能力更强，而是终于开始“讲人话”了。

GitHub：
https://github.com/fengzoey01-oss/skill-home

## 仓库介绍开头版

`skill-home` is a simple home entry for Codex skills.

It is built for one practical problem: once you install more and more skills, it becomes harder to remember which one to use for a given task.

Instead of forcing users to memorize skill names, `skill-home` lets them describe the goal in plain language, recommends the best-fit installed skill, and asks for confirmation before continuing.
